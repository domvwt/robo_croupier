============
Installation
============


Stable release
--------------

To install Robo Croupier, run this command in your terminal:

.. code-block:: console

    $ pip install robo_croupier

This is the preferred method to install Robo Croupier, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Robo Croupier can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/domvwt/robo_croupier

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/domvwt/robo_croupier/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/domvwt/robo_croupier
.. _tarball: https://github.com/domvwt/robo_croupier/tarball/master
